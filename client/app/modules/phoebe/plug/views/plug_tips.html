概述

介绍提示框的使用方法。

你可以在任何是有使用$.tips(info,type)向用户提示一些内容，比如在向服务器端请求一些表格数据并返回后，在回调函数中使用提示框的形式告知用户。

文档目录

实例

使用方法

$.tips(information,type)方法用于弹出提示框。

第一个参数information是提示框的内容。

第二个参数type是弹出框的类型，"success"将弹出绿色提示框，"error"将弹出红色提示框。

如果省略第二个参数，则默认弹出红色提示框。

	
<!--提示框示例代码-->
<button type="button" class="btn btn-success btn-sm" ng-click="successTip()">成功提示框</button>
<button type="button" class="btn btn-danger btn-sm" ng-click="failureTip()">失败提示框</button>

<script type="text/javascript">
    $scope.successTip = function (){
        $.tips('成功提示框是绿色的。弹出后将在3妙后消失！','success');
    };

    $scope.failureTip = function (){
        $.tips('失败提示框是红色的。弹出后将在3妙后消失！','error');
    }
</script>

提示框的样式是置顶的（始终现实在最外面）。
页面中同一时间只能显示一个提示框。
