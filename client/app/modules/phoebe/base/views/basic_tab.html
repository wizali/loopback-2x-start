使用方法

TAB页头：给<ul>标签添加class="nav nav-tabs"，将其中每个li > a的href属性设置为要显示代码块的id（例如：href="#home"），并为该a标签设置data-toggle="tab"属性。

TAB内容：放在一个class="tab-content"的<div>标签中，每一个TAB内容块用class="tab-pane fade"的<div>标签包起来，并将该<div>的id属性设置成其所对应的TAB页的id（例如：id="#home"）即可。

设置页面初始化要显示的tab，可以在TAB页头的<ul>中给相应的<li>添加.active样式，在TAB内容中将相应的<div>添加 .in .active样式。

API 接口

1、设置初始化TAB

设置初始化TAB除了使用上面介绍的给HTML标签加class样式以外，还可以使用JavaScript代码实现。从而可以操作整个TAB控件，比如添加监听事件，或者其他的事情。方法是给TAB页头设置id属性（例如：<ul class="nav nav-tabs" id="myTab">），然后使用js代码初始化TAB控件。如下：

	
<ul class="nav nav-tabs" id="myTab">
	<li><a href="#home"  data-toggle="tab">通话记录</a></li>
	<li><a href="#profile"  data-toggle="tab">电话本</a></li>
	<li><a href="#messages" data-toggle="tab">短信息</a></li>
	<li><a href="#settings" data-toggle="tab">微信</a></li>
</ul>
<script>
	$(function () {
	  //显示id="settings"的TAB
	    $('#myTab a:last').tab('show');
	})
</script>

2、监听TAB切换动作

监听TAB切换动作，类似于像<button>添加onClick事件。TAB控件提供两个鉴定属性，如下所示：

  1. hide.bs.tab (在当前TAB激活后触发)
  2. show.bs.tab (某个TAB即将激活前触发)

用任何一种方式监听，通过回调函数中的event参数获得当前TAB的标签对象和切换前TAB的标签对象。

	
$('a[data-toggle="tab"]').on('hide.bs.tab', function (e) {
	e.target //  被激活的TAB
	e.relatedTarget //  前一个激活的TAB
})
