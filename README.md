# My Application

一个node的测试demo，后台使用loopback，前台使用Angulars+bootstrap。

##build
```javascript
1.npn install -g strongloop
2.git clone https://github.com/wizali/loopback-2x-start.git
3.cd server
4.npm install
5.cd client
6.bower install
7.npm install
```
##start
```javascript
1.cd server && slc run
2.cd client && gulp
3.cd client && npm start
```
